== Description ==

<p>Are you worried about the spam?  AsrVerifyCode is easy and simple!Try it now!</a></p>
<p>AsrVerifyCode is an anti-spam for Wordpress comments.The plugin works when reviewers submit replys,and then check the Verify Code.If the code does not correct it cannot comment.</p>
<p>AsrVerifyCode can support safety login for you. When you login in, you must enter the verify code showing on the image.</p>

<br /><br /><br />
<p>讨厌垃圾评论？试试简单有效的AsrVerifyCode吧！</p>
<p>AsrVerifyCode是一个防垃圾评论的Wordpress插件。该插件在用户输入评论时需要输入随机验证码，提交后进行验证。如果验证码不正确则不能发表评论。</p>
<p>AsrVerifyCode也是一个安全登录的插件，当你需要登入时，你必须输入图片中的验证码，如果验证码不正确将不能登录。</p>
== Installation ==

<li>Upload the folder `/asr-verify-code/` to the `/wp-content/plugins/` directory</li>
<li>Activate the plugin through the 'Plugins' menu in WordPress</li>
